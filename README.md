# Student Management System (C++)

This is a simple console-based Student Management System written in C++.
It allows to add, view, search, update, and delete student records.
All data is stored in a text file (`students.txt`).

## Features
- Add new student
- Display all students
- Search by roll number
- Update marks
- Delete a student
- Save and load data from file

## How to Run
1. Compile the program:
   ```bash
   g++ main.cpp -o student_mgmt
